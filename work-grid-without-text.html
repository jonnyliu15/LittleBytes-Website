See what our students have accompished:

This Android app, written in Thunkable, is a fully interactive “catch” game, using monkey and banana Image Sprite components to demonstrate the student’s knowledge of mobile animation. The graphics are not original, but rather a creative amalgamation of PNG files, which students learn to manipulate, position, and find.

Hangman, although a popular game among children, also makes for a great programming lesson! This web application, written in HTML, CSS, and JavaScript, demonstrates conditional logic skills that are part of the “core four” tools we teach students; functions, loops, and variables come along as well.

One student saw a “galaxy shooters” game on CoolMathGames, and wanted to create his own, distractions free version with just asteroid and meteor Image Sprites in Thunkable. This game is another creative application of the animation, gaming, and graphics skills LittleBytes students learn, to make their work usable and clean.

The LittleBytes curriculum has a major emphasis on using technology positively, with balance, and for social good. At the end of session 1, Web Development students were asked to create websites on a social issue of their choice, accurately sourcing their information and presenting it in an aesthetically pleasing way.

While the majority of our Python Logic Fundamentals class focuses on algorithms and arithmetic thinking, the LittleBytes curriculum includes some of the more fun applications of Python as well. Here, a student’s code for a from-scratch stick figure is shown. It is written in the Turtle library for Python 2.7, which was originally developed in the 1900s for the purpose of teaching children computing logic.

LittleBytes teaches students the skills they need for jobs that don’t even exist today - one of these skills is an understanding of Artificial Intelligence. AIBuddy, shown in the picture, is an Android app that uses the Microsoft Image Recognition API to interpret photos. The students learned how to parse the API result into natural language for the user to read.
